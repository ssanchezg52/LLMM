Globo App

Consiste en una aplicacion que, cuando le das a un boton, empiezan a caer globos de forma aleatoria.

Formulario JS

Es un formulario que cuando le das a guardar, toda la informacion que ha sido rellena en el formulario se muestra en un textfield que aparece cuando le das al boton.

Twitter

Es una pagina inspirada en twitter donde tienes que seleccionar un archivo xml, para que aparezca toda la informacion que contiene. Descarga el archivo y despues tienes que seleccionar el archivo dentro de la pagina.

Tiempo

Esta pagina coge la informacion de un JSON y lo mete en la pagina pudiendose ver la temperatura, ciudad, viento, humedad y el tiempo en los 3 dias siguientes

API Futbol

Esta pagina te dice los partidos de los ultimos dias y incluso puedes ver un highlights, puedes comprobar si esta terminado.. etc. Incluso tiene un buscador por si te interesa uno de los partidos. La informacion fue obtenida de una API de futbol.
